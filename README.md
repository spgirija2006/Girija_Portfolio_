# New Workspace Setup

This workspace contains a simple responsive HTML/CSS starter page.

## Files
- `index.html` — main page
- `styles.css` — styling

## Run locally
1. Open `index.html` in your browser.
2. Or use a local server (recommended):
   - Visual Studio Code: Install Live Server extension and click "Go Live".
   - Command line: `python -m http.server 5500` (from project folder) and open `http://localhost:5500`.

## Enhance
- Add `script.js` for JavaScript interaction.
- Add menu, animations, form, etc.
